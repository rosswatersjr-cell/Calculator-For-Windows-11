# Calculator-For-Windows-11
Created Using Python 3.14.3, customtkinter Sympy And mpmath. This Calculator Has 2 Language Choices (English Or Spanish). 
Right Click The Display For All Available Options. Also Right Click Buttons For Help. View ctkClculator.png File For Appearance.
The Support png Files And json File Should Be Placed In The _internal Folder For The Executable per Code.

How to Fix Using Auto-Py-To-Exe For CTkColorPicker:  
1. Add Hidden Import: In the Advanced section, find --hidden-import and add CTkColorPicker. 
2. Collect All Submodules: In the Advanced section, locate the --collect-all field and enter CTkColorPicker. 
   This ensures all internal files are included.
