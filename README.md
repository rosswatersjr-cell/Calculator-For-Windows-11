# Calculator-For-Windows-11
Created Using Python 3.14.3, customtkinter And Sympy. This Calculator Has 2 Language Choices (English Or Spanish). 
Right Click The Display For All Available Options. Also Right Click Buttons For Help. View png File For Appearance. 
How to Fix Using Auto-Py-To-Exe For CTkColorPicker 
Add Hidden Import: In the Advanced section, find --hidden-import and add CTkColorPicker.
Collect All Submodules: In the Advanced section, locate the --collect-all field and enter CTkColorPicker.
This ensures all internal files are included.
